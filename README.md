# 📊 freeCodeCamp Data Analysis with Python Projects

This repository contains my solutions for the **freeCodeCamp Data Analysis with Python Certification**. Each project demonstrates the use of Python and popular data analysis libraries to solve real-world data processing and visualization problems.


## 🚀 Projects

| Project | Description | Technologies |
|---------|-------------|--------------|
| **Mean-Variance-Standard Deviation Calculator** | Computes statistical measures (mean, variance, standard deviation, min, max, and sum) for a 3×3 matrix using NumPy. | Python, NumPy |
| **Demographic Data Analyzer** | Analyzes census demographic data to answer statistical questions using Pandas. | Python, Pandas |
| **Medical Data Visualizer** | Cleans and visualizes medical examination data using categorical plots and heatmaps. | Python, Pandas, Matplotlib, Seaborn |
| **Page View Time Series Visualizer** | Visualizes freeCodeCamp forum page views using line charts, bar charts, and box plots. | Python, Pandas, Matplotlib, Seaborn |
| **Sea Level Predictor** | Predicts future sea level rise through 2050 using linear regression. | Python, Pandas, Matplotlib, SciPy |

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/freecodecamp-data-analysis-with-python.git
cd freecodecamp-data-analysis-with-python
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

> Each project also contains its own `requirements.txt` if you prefer to install dependencies individually.

---

## ▶️ Running a Project

Navigate to the desired project folder:

```bash
cd demographic-data-analyzer
```

Run the project:

```bash
python main.py
```

Run the tests:

```bash
python test_module.py
```
