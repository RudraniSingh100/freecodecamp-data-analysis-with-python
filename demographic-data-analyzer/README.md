# Demographic Data Analyzer

This project is part of the **freeCodeCamp Data Analysis with Python Certification**.

## Overview

Analyze the Adult Census Income dataset using **Pandas**.

The project answers questions such as:

- Number of people by race
- Average age of men
- Percentage with Bachelor's degrees
- Income comparison based on education
- Minimum working hours
- Country with the highest percentage of people earning over \$50K
- Most common occupation among high-income individuals in India

## Technologies

- Python
- Pandas

## Dataset

- adult.data
- Source: UCI Machine Learning Repository

## Run

```bash
pip install -r requirements.txt
python main.py
```
